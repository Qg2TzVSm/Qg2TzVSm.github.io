列表

1、无序列表

在文字前加上- 或*即可（中间最好有3个空格）。

  • a
  • b
  • c

2、有序列表

在文字前加上数字. 即可(使用的数字并不会影响输出到html上的显示结果)。

  1. fff
  2. fff
  3. fff

This is [an example](http://example.com/ "Title") inline link.

This is an example inline link. 

[This link](http://example.net/) has no title attribute.

本机的资源，使用相对路径。

See my [About](/about/) page for details.

链接还可以使用[][id] 这样的形式在文章的其它地方加上：

[id]: https://cdn.wiki.dengekionline.com/files/attachment/000/237/448/ref_upload.jpg "文章其它地方"

上面的地址也可以用方括号包起来：

[id]: <http://example.com/>  "Optional Title Here"

title属性可以放到下一行，这样在链接比较长的时候比较好看：

[id]: http://example.com/longish/path/to/resource/here
"Optional Title Here"
